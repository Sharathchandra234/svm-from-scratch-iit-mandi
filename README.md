# SVM From Scratch (IIT Mandi)

Support Vector Machine (SVM) implementation from scratch for Bank Marketing Classification, developed as an academic project during the IIT Mandi Minor Degree in Data Science & Artificial Intelligence.

## Overview

This project demonstrates the implementation of a Support Vector Machine (SVM) using Sequential Minimal Optimization (SMO) to solve a binary classification problem on the Bank Marketing Dataset. The project focuses on understanding the underlying mathematics and optimization techniques behind SVMs rather than relying solely on pre-built libraries.

## Key Features

* Implemented Support Vector Machine (SVM) from scratch.
* Utilized Sequential Minimal Optimization (SMO) for training.
* Performed data preprocessing and feature engineering.
* Applied classification techniques on the Bank Marketing Dataset.
* Evaluated model performance using Accuracy and F1-Score.
* Documented findings and implementation details.
